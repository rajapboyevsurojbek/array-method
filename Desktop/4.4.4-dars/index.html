JavaScript Array Methods

  • push()

    Array'ga yangi element qo'shadi.

    arr.push(4); // [1, 2, 3, 4]

  • pop()

    Array'dan oxirgi elementni olib tashlaydi. 

    arr.pop(); // [1, 2]

  • shift()

    Array'dan birinchi elementni olib tashlaydi. 

    arr.shift(); // [2, 3]

  • unshift()

    Array'ga yangi elementni birinchi o'ringa qo'shadi. 

    arr.unshift(0); // [0, 1, 2, 3]

  • slice()

    Array'dan yangi qismini olish uchun ishlatiladi. 

    arr.slice(1, 3); // [2, 3]

  • concat()

    Bir yoki bir nechta array'ni birlashtiradi. 

    arr1.concat(arr2); // [1, 2, 3, 4]

  • join()

    Array elementlarini birlashtirib, stringga aylantiradi. 

    arr.join(', '); // "1, 2, 3"

  • map()

    Array elementlarini qayta ishlash uchun yangi array qaytaradi. 

    arr.map(x => x * 2); // [2, 4, 6]

  • filter()

    Array elementlarini saralash uchun ishlatiladi. 

    arr.filter(x => x > 2); // [3]

  • reduce()

    Array elementlarini bitta qiymatga yig'ish uchun ishlatiladi. 

    arr.reduce((a, b) => a + b, 0); // 6

  • find()

    Array'dan ko'rsatgan shartlarga javob beradigan birinchi elementni topadi. 

    arr.find(x => x > 2); // 3

  • some()

    Array'da shart bajarilsa `true` qaytaradi, aks holda `false`. 

    arr.some(x => x > 2); // true

  • every()

    Barcha elementlar shart bajarilsa `true` qaytaradi, aks holda `false`. 

    arr.every(x => x > 0); // true

  • sort()

    Array'ni tartibga solish uchun ishlatiladi. 

    arr.sort(); // [1, 2, 3]

  • reverse()

    Array elementlarini o'rnatishni teskari qiladi. 

    arr.reverse(); // [3, 2, 1]
